GETTING THE Kakaotalk ID

1. Open the Kakaotalk app on your smart phone. *(Login if you didn't login yet)

2. In your Kakaotalk app, press the More Option Button. ( 3 dot shown on the image below)

3. Press My Profile. ( Primary Photo with your name shown on the image below )

4. If you have set your Kakaotalk ID before you will see your Kakaotalk ID in the image below. Or else press the KakaoTalk ID to set your ID.

5. Enter the desired Kakaotalk ID, then press OK. ( Kakaotalk ID is unique to everyone )

6. Good Job! You have properly set your kakaotalk ID.
